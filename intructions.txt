1. Create project using available template
`bunx create-expo -t https://github.com/EvanBacon/expo-router-forms-components .` 

2.
`export BUN_IOS_SKIP_XCODE_CHECK=1`
`bun start`

3. Shortcut in emulator
- Press 'R': to reload
- Press 'Ctrl + shift + a': to change dark/light mode 

4. 
      [
      "expo-router",
      {
        "origin": "https://api.routes.expo.app"
      }
      ],


5. 
`bun add jsonwebtoken`

6. docker-compose.yml
`docker compose up`

7. 
`bun add drizzle-orm postgres`

8. restart typescript server

9.
`bun add drizzle-kit`

10.
`bun add env`

11.
`bun add @types/jsonwebtoken`

12.
`bun add expo-secure-store`
